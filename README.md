Online Examination System (2026)
A modern, responsive, and secure web-based portal for conducting student examinations. 

Features :
1. Authentication & Security
Secure Login: Requires a valid @gmail.com address and a strong password (8+ characters, including uppercase, lowercase, and numbers).

Proctoring Logic:

Detects and alerts the user if they attempt to switch browser tabs.

Disables right-click and text copying to prevent cheating.

Timer System: 10-minute countdown per exam with automatic submission upon expiration.

2. Exam Engine
Multiple Subjects: Supports JavaScript, Python, and Java.

Dynamic Shuffling: Questions and answer options are randomized for every session using the Fisher-Yates shuffle algorithm.

Uniform Assessment: Each subject contains 5 distinct questions.

3. User Interface :
Home Page: Contains a detailed examination guideline and an interactive "Enter Portal" button.

Global Profile: A floating interactive profile circle available across all dashboard tabs, showing the user's name and email.

4. Performance Analysis
Real-time Grading: Results are calculated immediately after submission.

Passing Criteria: Status is marked as Passed if the score is above 35%, otherwise Failed.

Score Persistence: Uses localStorage to save and display the highest score achieved for each subject.
